# 🎬 MovieVerse

## 📌 Project Description

MovieVerse is a movie explorer web application that allows users to discover and search for movies using real-time data from the OMDb API. 
The application provides an interactive and user-friendly interface to explore movies and view their details.

## 🚀 Features

* 🔍 Search movies by title
* 🎯 Filter movies based on release year
* 🔃 Sort movies by year (ascending/descending)
* 🎬 Dynamic movie display using API data
* 🖼️ Custom “No Image” UI for missing posters
* 📱 Fully responsive design
* ❤️ Add / Remove movies from Favorites
* ⭐ View favorite movies
* 🚫 Prevent duplicate favorites
* 💾 Favorites stored using Local Storage
* ⬅ Back button to return to main page
* 🌙 Dark / Light mode toggle
* ⚡ Debounced search for better performance

## 🌐 API Used

* OMDb API (Open Movie Database)

## 💡 Key Concepts Used

* Fetch API for data retrieval
* JavaScript Higher Order Functions:

  * map()
  * filter()
  * sort()
* DOM manipulation
* Event handling
* Local Storage (for saving favorites)  
* Debouncing (performance optimization)
